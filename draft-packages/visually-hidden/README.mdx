Use VisuallyHidden when you want to visually hide an element, but still 
make it available to screen reader users. 

If the hidden element is focusable (e.g. a Link or Button),
then when this element receives focus it will be visible. This is useful for 
[Skip to main content](https://www.w3.org/TR/2013/NOTE-WCAG20-TECHS-20130905/G1) links.

<WhenToUseAndWhenNotToUse>

<WhenToUse>

* Use VisuallyHidden when meaning is apparent visually but may not be apparent to screen reader users.

</WhenToUse>

<WhenNotToUse>

* For content that is beneficial for both sighted users and those using assisted technologies.
* Do not repeat content. Verbose cues or instructions are generally a nuisance for screen reader users.

</WhenNotToUse>

</WhenToUseAndWhenNotToUse>
